# RPA Web Automation -- Process Optimization

Automação profissional desenvolvida em Python para otimizar processos
operacionais realizados em sistema web corporativo.

------------------------------------------------------------------------

## Visão Geral

Este projeto implementa uma solução de RPA (Robotic Process Automation)
utilizando Selenium para automatizar tarefas repetitivas executadas
manualmente em ambiente web.

A automação reduz tempo operacional, minimiza erros humanos e padroniza
a execução do processo.

------------------------------------------------------------------------

##  Problema Resolvido

Processos manuais exigiam:

-   Login repetitivo em sistema web
-   Navegação constante entre páginas
-   Preenchimento manual de formulários
-   Execução sequencial de rotinas operacionais

Impactos identificados:

-   Perda de produtividade
-   Alto risco de erro humano
-   Dependência operacional

------------------------------------------------------------------------

## Solução Implementada

A automação realiza:

-   Login automatizado
-   Navegação estruturada no sistema
-   Preenchimento inteligente de campos
-   Execução automática do fluxo operacional
-   Leitura e tratamento de dados externos
-   Configuração dinâmica via arquivo `.ini`

------------------------------------------------------------------------

## Stack Tecnológica

-   Python

## Como Executar

Clone o projeto:

    git clone https://github.com/kaiqueazevedo/Automation_rpa_ndi.git

Instale as dependências:

    pip install -r requirements.txt

Execute:

    python main.py
